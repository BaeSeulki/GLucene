# GLucene
## Lucene desktop searcher  基于lucene的中文全文桌面搜索引擎
需要导入的依赖包[百度网盘下载](http://pan.baidu.com/s/1gfJ5dVP)
目前的依赖包版本：
lucene-4.0
中文分词包：IKAnalyzer2012_FF
文本解析包：POI-3.15
xml解析包：xmlbeans-2.60
